# website_activity_homepage

~~ちゃんとウェブサイトを作ろうとするのは初めてだよ。既にgitがわからん過ぎてつらい！！！~~

自分の部活を紹介するホームページです。
私のセキュリティパワーを加味した結果、ここに貼ってあるデータは大体全てがダミーです。

また、外部に自分のコードを公開することもほとんどなかったので、他の環境でも動かせるように調整できているかはわかりません。

技術やフレームワークとしては、ザックリ以下のものを使っています。

- docker
- node.js
- express
- ejs
- postgreSQL

現在データベース接続用のapiが何一つできていないため、これから作成します。
いい感じになるように行けるところまで頑張ります。
